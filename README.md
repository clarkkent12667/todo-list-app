# ✅ To-Do List App

A simple and responsive **To-Do List App** built with **React, Node.js, and PostgreSQL**, allowing users to **create, update, and manage tasks efficiently**. 🚀

## 🔥 Features
- ✅ **User Authentication** (Signup, Login, Logout)
- 📝 **Add, Edit, Delete Tasks**
- 📅 **Task Categories & Due Dates**
- 🎨 **Mobile-Friendly & Responsive Design**
- 🌙 **Dark Mode Support** *(if added)*
- 📊 **Task Completion Status**
- 🚀 **Deployed with Replit Always On**

---

## 🛠️ Tech Stack
### **Frontend:**
- React.js (or Next.js if applicable)
- Tailwind CSS (or plain CSS)

### **Backend:**
- Node.js + Express.js
- PostgreSQL (with Prisma ORM)

### **Deployment:**
- **Frontend:** Replit (Always On)  
- **Backend:** Replit (Always On)  
- **Database:** Supabase / ElephantSQL *(if applicable)*
